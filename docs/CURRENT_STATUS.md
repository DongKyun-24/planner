# Current Status

마지막 정리일: 2026-05-06

## 지금 되는 것

- 웹은 Vite + React + Supabase 기반이다.
- 일정은 날짜별 리스트/달력에서 보고, 탭별 카테고리로 나눠 관리한다.
- 빠른 일정 추가/수정 모달에서 내용, 시작일, 종료일, 반복, 시간, 알림을 설정한다.
- 종료일은 `없음`, 날짜 지정, `계속` 반복을 구분한다.
- `계속`은 종료일 없는 반복으로 저장한다.
- 반복 설정은 종료일을 지정하거나 `계속`을 눌렀을 때만 보인다.
- 시간 설정은 시작/종료 블록으로 나뉜다.
- 종료 시간은 기본 비활성 상태이며 `설정 안 함`으로 보인다.
- 시간 선택은 오전/오후, 시, 분을 커스텀 스크롤 패널에서 고른다.
- 분은 `00`부터 `59`까지 선택할 수 있다.
- 선택된 시간 옵션은 hover가 다시 덮지 않도록 정리했다.
- 알림은 `알림 없음`, `정시`, `5분 전`, `10분 전`, `30분 전`을 지원한다.
- 반복 일정은 유한 반복과 종료일 없는 반복을 모두 저장 흐름에 연결했다.

## 이번 작업에서 정리한 것

- 시간 선택 select 기본 화살표를 제거했다.
- 시간 옆에 중복으로 보이던 날짜 표시를 제거했다.
- 시간 선택 UI를 select에서 블록 + 인라인 패널 구조로 바꿨다.
- 시간 패널 높이와 hover 스타일을 조정했다.
- 종료일 `계속` 버튼을 빠른 일정 모달에 추가했다.
- 종료일이 없을 때 반복 섹션이 먼저 뜨지 않도록 했다.
- `계속` 반복이 저장 과정에서 날짜 반복으로 바뀌지 않게 했다.
